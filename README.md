# ⚡ High-Performance Matching Engine

A C++ Order Book matching engine designed for low-latency trading environments.

## 🚀 Key Features
* **Limit Order Book:** Efficient management of Buy/Sell orders in memory.
* **Anti-DDoS / Rate Limiting:** Algorithmic protection against "Quote Stuffing" attacks.
* **Performance:** Optimized C++ code processing input streams via Unix Pipes.

## 🛠 Stack
* **Language:** C++ (GCC optimized)
* **Tooling:** Python (for load generation and attack simulation)
* **Ops:** Docker
